# Ophira Landing Page

Vanilla HTML/CSS/JavaScript version of the Ophira landing page.

## Files Structure

- `index.html` - Main HTML file with all sections
- `styles.css` - Custom CSS styles
- `script.js` - JavaScript for interactivity (mobile menu, scroll effects, FAQ accordion, testimonials carousel, pricing toggle)
- `assets/` - Images and SVG files

## Features

- Responsive design optimized for mobile, tablet, and desktop
- Mobile menu toggle
- Navbar scroll effect (changes appearance on scroll)
- FAQ accordion functionality
- Testimonials carousel
- Pricing toggle (Monthly/Yearly)
- Smooth transitions and animations

## How to Use

Simply open `index.html` in a web browser. All dependencies (Tailwind CSS, FontAwesome, Google Fonts) are loaded via CDN.

## Dependencies (loaded via CDN)

- Tailwind CSS
- FontAwesome 6.4.0
- Google Fonts (DM Sans, Anybody)

No build process or framework required - pure vanilla HTML/CSS/JS!
